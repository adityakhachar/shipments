# Task: Designing an Automated Email Scheduling API


## Installation
1. Clone the repository
```bash
    git clone https://github.com/adityakhachar/shipments.git
```

2. Install the package
```bash
    git install
```
3. Start the server
```bash
    nodemon index.js
```
